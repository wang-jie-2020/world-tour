梦幻写实的 3D 微缩地球乐园风格，融合了数字孪生地球、游戏美术、奇幻生态星球、世界地标百科界面等元素。

具体特点：

主体形式：巨大的 3D 地球悬浮在画面中心，类似一个被改造成乐园或博物馆的微缩星球。
内容主题：地球表面不再是普通地貌，而是布满世界著名地标、历史遗迹、自然景观和城市建筑，如长城、金字塔、泰姬陵、东方明珠、悉尼歌剧院等。
视觉质感：整体偏电影级 3D 渲染，细节丰富，建筑和山川都有明显的立体起伏，具有精致游戏场景或高端动画宣传片质感。
色彩氛围：以浅蓝、青绿、暖金、柔白为主，光线明亮通透，带有柔和梦幻感，而不是强烈魔幻感。
光影效果：顶部和右上方有柔和阳光照射，地球边缘带透明大气层与白色光晕，营造温暖、神圣、探索感。
构图方式：竖屏短视频封面式构图，地球占据中心大部分画面，顶部标题、底部信息卡和分类导航栏形成完整的应用界面视觉。
UI 设计：半透明玻璃拟态风格，底部有筛选栏，中间有地标信息卡片，类似“互动百科 / 探索类 App / 知识短视频界面”。
文字风格：标题字体偏优雅、装饰性强，带微弱发光和花纹边饰，增强童话乐园般的梦幻感。
整体气质：精致、治愈、明亮、梦幻、百科探索感，像是“如果地球是一座主题乐园”的数字艺术海报。
