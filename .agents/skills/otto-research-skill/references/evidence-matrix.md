## Rules

1. Reposts and summaries of the same origin count as one source.
2. A claim is `verified` only when at least two independent high-quality sources support it.
3. Product-level claims require at least one official source (official docs or official repo).
4. If evidence conflicts, mark as `disputed` and record what would resolve it.
5. If only one source exists, keep `unverified` and do not present as final fact.

## Minimum Report Mapping

- Every key conclusion in the final report must map to at least one `Claim ID`.
- Every `Claim ID` should map to explicit links in source list.
- Keep fact vs inference separated in notes.
